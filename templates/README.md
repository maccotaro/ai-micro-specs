# ドキュメントテンプレート

新規ドキュメント作成時に使用するテンプレート集です。

## 📄 テンプレート一覧

- `service-overview-template.md` - サービス概要テンプレート
- `api-specification-template.md` - API仕様書テンプレート
- `database-design-template.md` - データベース設計テンプレート
- `adr-template.md` - アーキテクチャ決定記録（ADR）テンプレート

## 使用方法

1. 適切なテンプレートをコピー
2. ファイル名を変更
3. 内容を記述
4. カテゴリのREADMEにリンクを追加
